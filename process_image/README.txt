For compiling program you should execute following commands:
chmod +x compile.sh
./compile.sh

For execute program:
./process_image.exe ./path/to/png_file.png

For example:
./process_image.exe ./images/Lena.png
As result output image is saved in the same folder in file with suffix "_out.png". In that case in file with path ./images/Lena.png_out.png
